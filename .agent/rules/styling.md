# Styling & Animations
- **Tailwind CSS**: Use Tailwind for all styling. Follow the established design system tokens in `tailwind.config.cjs`.
- **Animations**: Use `framer-motion` for complex transitions and animations. Prefer the `Reveal` or `PageTransition` components for standard entrance/exit animations.
- **Micro-interactions**: Implement subtle hover effects and transitions for interactive elements.
- **Theming**: Respect the theme provided by `ThemeContext`.
